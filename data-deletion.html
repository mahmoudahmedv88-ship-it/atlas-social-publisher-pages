Data Deletion Instructions

Last updated: July 2026

ATLAS Social Publisher stores minimal data locally on your device. This page explains how to request deletion of any data associated with your use of this application.

What Data Is Stored

  • Your TikTok account identifier (open_id) - stored locally only
  • OAuth access token and refresh token - stored locally only
  • Token expiry dates and granted permissions - stored locally only
  • Connection timestamps and authorization status - stored locally only

No data is stored on any external server. All data exists only on your local device.

How to Delete Your Data

Option 1: Disconnect via TikTok Settings (Recommended)

  1. Open the TikTok app on your device
  2. Go to Profile → Settings and Privacy
  3. Select Security → Apps and Permissions
  4. Find ATLAS Social Publisher in the list
  5. Tap Revoke Access

This will immediately invalidate all tokens and disconnect the application from your TikTok account.


Option 2: Delete Local Data Files

Since all data is stored locally on your device, you can delete it by:

  1. Locating the ATLAS Social Publisher application folder on your device
  2. Deleting the .env file or clearing the token storage
  3. Removing any database files associated with the application

Request Data Deletion via GitHub

If you need assistance with data deletion or have concerns about stored data, you can submit a request by opening an issue on our GitHub repository with the title "Data Deletion Request".

We will respond within 30 days and confirm that all associated data has been removed.

TikTok Data

For data held by TikTok directly, please refer to TikTok's Privacy Policy and their data deletion procedures.
